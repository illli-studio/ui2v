# Example Roles

## README priority order

1. **Hero AI Launch**: first-impression trailer. Must look premium immediately.
2. **Product Showcase**: concrete product launch use case. Users should imagine swapping in their own product.
3. **Render Lab**: capability reel. Show particles, data, pseudo-3D, runtime/export workflow.
4. **Commerce Command Center**: business-data/ops scenario. Show realistic value beyond toy visuals.

## Naming

- Use lowercase hyphenated folder names for template examples.
- Use descriptive runtime JSON names under `examples/runtime-core`, e.g. `uiv-runtime-commerce-command-center.json`.
- IDs should match the concept and stay stable enough for README links.

## What makes an example attractive

- It has a target user: founder, devtool team, data storyteller, AI agent workflow.
- It has a finished-video use case: README hero, launch clip, dashboard report, social ad.
- It has copy that explains value without needing external context.
- It can be rendered and converted to a short GIF under 3 MB.
