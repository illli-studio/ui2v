# Example Roles

## Maintained examples

1. **Basic Smoke**: smallest JSON render sanity check.
2. **Library Timeline**: multi-library example with visible time-based dependency beats.
3. **Access Media**: local `access/` folder example for image, video, waveform, and muxed audio.
4. **Runtime Storyboard**: runtime-core example for segments, transitions, camera metadata, and inspection.

## Naming

- Use lowercase hyphenated folder names for template examples.
- Use descriptive runtime project folders such as `examples/runtime-storyboard`.
- IDs should match the concept and stay stable enough for README links.

## What makes an example attractive

- It has a target user: founder, devtool team, data storyteller, AI agent workflow.
- It has a finished-video use case: README hero, launch clip, dashboard report, social ad.
- It has copy that explains value without needing external context.
- It can be rendered and converted to a short GIF under 3 MB.
