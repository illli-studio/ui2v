# Paper Route Planner

[English](README.md)

一个维护中的真实库示例，由 Paper.js 负责矢量路线、平滑曲线、检查点和移动点。

```bash
node packages/cli/dist/cli.js preview examples/paper-route/animation.json --pixel-ratio 1
node packages/cli/dist/cli.js render examples/paper-route/animation.json -o .tmp/examples/paper-route.mp4 --quality high
```
