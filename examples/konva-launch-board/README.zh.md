# Konva Launch Board

[English](README.md)

一个维护中的真实库示例，使用 Konva 节点组装带动画的产品发布故事板。

```bash
node packages/cli/dist/cli.js preview examples/konva-launch-board/animation.json --pixel-ratio 1
node packages/cli/dist/cli.js render examples/konva-launch-board/animation.json -o .tmp/examples/konva-launch-board.mp4 --quality high
```
