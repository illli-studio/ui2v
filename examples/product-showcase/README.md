# Product Showcase Example

Professional product showcase animation.

## Features

- Gradient background
- Logo animation
- Title with scale effect
- Subtitle fade in
- Call-to-action with bounce

## Usage

```bash
node ../../packages/cli/dist/cli.js validate animation.json --verbose
node ../../packages/cli/dist/cli.js render animation.json -o ../../out/product-showcase.mp4 --quality high
```
