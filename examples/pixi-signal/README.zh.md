# PIXI Signal Rescue

[English](README.md)

一个维护中的真实库示例，由 PixiJS 驱动移动雷达粒子和扩散救援环。

```bash
node packages/cli/dist/cli.js preview examples/pixi-signal/animation.json --pixel-ratio 1
node packages/cli/dist/cli.js render examples/pixi-signal/animation.json -o .tmp/examples/pixi-signal.mp4 --quality high
```
