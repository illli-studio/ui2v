# Type Systems Map

A technical infographic that exercises typography-related libraries: `KaTeX` prepares formula markup, `Iconify` loads an offline MDI icon, and `opentype.js` contributes glyph/vector path tooling.

## Preview

```bash
node packages/cli/dist/cli.js preview examples/type-systems-map/animation.json --pixel-ratio 1
```

## Render

```bash
node packages/cli/dist/cli.js render examples/type-systems-map/animation.json -o .tmp/examples/type-systems-map.mp4 --quality high
```
