# Type Systems Map

一个技术信息图例子，用来展示字体相关库：`KaTeX` 生成公式 markup，`Iconify` 加载离线 MDI 图标，`opentype.js` 参与字形/矢量路径绘制。

## 预览

```bash
node packages/cli/dist/cli.js preview examples/type-systems-map/animation.json --pixel-ratio 1
```

## 渲染

```bash
node packages/cli/dist/cli.js render examples/type-systems-map/animation.json -o .tmp/examples/type-systems-map.mp4 --quality high
```
